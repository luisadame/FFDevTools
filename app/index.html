Firefox Developer Tools

What are the Firefox developer tools?

These are a set of tools that allow you edit, examine, debug, and obtain information on the application you are working on. These tools are available only in the Firefox browser.

Organization

As everything, FF devtools are organized in a few categories:

  • The Core Tools
  • The less frequent tools
  • Connecting devtools
  • Debugging browser tools
  • Tools for tools

How To?

The developer tools can be opened in a few different ways, but mainly:

  • Right clicking the page and click on the "Inspect Element" option.
  • Press the F12 key.
  • Press the combination Ctrl+Shift+I.
  • Click on the hamburger button on the top right corner, click on the Web Developer menu, and click on "Toggle Tools".
