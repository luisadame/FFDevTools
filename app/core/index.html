The Core Tools

The page inspector

With the page inspector you can view and change page content and layout. You can get vital information from the page visual behaviour such as the box model, animations and grid layouts.

Functionalities

  • Examine and edit HTML
  • Show and edit the box model of an element
  • Inspect and select colors
  • Change the position of elements
  • Edit fonts
  • Visualize transforms
  • Examine and edit CSS
  • Examine event listeners
  • Work with animations
  • Edit CSS filters
  • Edit CSS shapes
  • View background images
  • Use the inspector from the web console
  • Examine CSS grid layouts

Examine and edit HTML

As we open the developer tools the first thing we encounter is the html pane which is positioned on the left side of the tools.

We'll cover first the outer elements of the pane and then go into the most important element. At the bottom we can see the breadcrumbs section from where the hierarchy of an element is displayed.

Also at the top there is a toolbar with three options:

  • Add a new node
  • Search the html
  • Grab color from page
The options are pretty self explained. The "add a new node" button allow us to create a new node in the html tree. The "Search the html" gives us the option to look for an element in the html tree. Lastly, the "grab a color from the page" is a tool for taking the color of a pixel from the page we are inspecting.

The last element is one of the most important for this tool. The HTML tree shows the elements of a web page and their content. You can edit the html of a site using this tool. For example: double clicking on an element makes it editable and we can change that html.
